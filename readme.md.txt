# Simulador Torricelli 🚦📈

Este é um aplicativo web interativo que simula a relação entre velocidade e espaço com base na equação de Torricelli. Ideal para estudos de engenharia de tráfego, física ou análise de comportamento veicular.

## 🔧 Funcionalidades

- Entrada de parâmetros globais: aceleração, desaceleração e velocidade regulamentar
- Inserção de múltiplos pontos com posição e velocidade
- Suporte a configurações como ondulações, semáforos, equipamentos eletrônicos
- Cálculo automático de pontos intermediários com base na equação de Torricelli
- Inserção de placas R-19 com velocidade constante
- Gráfico interativo de Velocidade vs Espaço (Chart.js)
- Tabela dinâmica com todos os pontos
- Consulta interativa por posição ou velocidade
- Exportação dos dados para CSV
- Impressão do gráfico e da tabela com layout limpo
